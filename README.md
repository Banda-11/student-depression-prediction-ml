# 🎓 Student Depression Prediction using Pattern Recognition

This project implements a Machine Learning system to predict student depression based on academic and lifestyle factors.

### 📊 Project Overview
The model analyzes various features such as:
* **Academic:** CGPA, Study Hours.
* **Lifestyle:** Sleep Duration, Physical Activity, Social Media Usage.
* **Demographics:** Age, Gender, Department.

### 🛠️ Modeling & Methodology
The project compares two classification algorithms:
1. **Logistic Regression** (Baseline model).
2. **Random Forest Classifier** (For higher accuracy and non-linear patterns).
